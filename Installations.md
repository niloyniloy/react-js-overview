# Installations

Follow those steps to install react in your pc. First make sure Node.js is installed in your PC.

1. Install React via Vite
```bash
npm create vite@latest your-folder-name -- --template react
```

2. If install in same/current directory
```bash
npm create vite@latest . -- --template react
```

3. Go to that folder 
```bash
cd your-folder-name
```


4. Then npm install 
```bash
npm install 
```
